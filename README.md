# 🎨 Figma Test Site

Адаптивный сайт, свёрстанный по макету из Figma. Чистая вёрстка для демонстрации навыков работы с дизайн-макетами.

## ✨ Функциональность

- 📐 Точная вёрстка по макету Figma
- 📱 Адаптивный дизайн (мобильные, планшеты, десктоп)
- 🎨 Чистые и валидные HTML/CSS
- 🖼️ Оптимизированные изображения
- 🔗 Доступ по прямой ссылке

## 🛠️ Технологии

- HTML5
- CSS3 (Flexbox/Grid)
- GitHub Pages

## 🚀 Демо

[Открыть сайт](https://nettbliss.github.io/figma_test_site/)

## 📖 Как использовать

1. Открой [демо-сайт](https://nettbliss.github.io/figma_test_site/)
2. Изучи вёрстку на разных устройствах (нажми F12 → режим телефона)
3. Сравни с оригинальным макетом Figma
